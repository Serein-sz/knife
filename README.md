# Knife 语言解释器

## 项目简介
Knife 是一个轻量级的脚本语言解释器，使用 Go 语言实现。

## 安装说明
1. 确保已安装 Go 1.23.3 或更高版本
2. 克隆本项目：
   ```
   git clone https://github.com/Serein-sz/knife.git
   ```
3. 进入项目目录并构建：
   ```
   cd knife
   go build
   ```

## 使用示例
```
// 示例代码
print("Hello, Knife!")
```
